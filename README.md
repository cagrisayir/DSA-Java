### Operations

* **push(item)**: Adds an item on top of the stack **O(1)**
* **pop()**: removes the item on the top. **O(1)**
* **peek()**: returns the item on the top without removing it. **O(1)**
* **isEmpty()**: tells us stack is empty or not. **O(1)**
